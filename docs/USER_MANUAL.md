# Руководство пользователя программы «Объединение Excel файлов»

Это инструкция для программы, которая помогает собирать данные из нескольких Excel-таблиц в один красивый отчёт с автоматической раскраской и графиками.

---

## Что делает эта программа?

Представь, что у тебя есть несколько отчётов Excel — по одному с каждого прибора или узла измерения. В каждом файле — данные за разный период. Тебе нужно собрать всё в один файл и посмотреть общую картину.

Программа делает это автоматически. Она умеет:

1. **Загружать** несколько файлов Excel одновременно
2. **Выбирать** нужные параметры (расход, давление, температура и т.д.)
3. **Фильтровать** данные по временному интервалу
4. **Объединять** всё в одну таблицу с правильными заголовками
5. **Раскрашивать** ячейки по значению: норма — зелёный, нарушение — красный
6. **Добавлять стрелки** ↑↓ рядом со значениями, которые вышли за допустимый диапазон
7. **Создавать Dashboard** — отдельный лист с наглядной сводкой по дням

### Левая колонка — «Файлы и Время»

Здесь ты управляешь тем, **что** обрабатывать.

- **Кнопка «Добавить файлы»** — открывает диалог выбора файлов (`.xlsx` или `.xls`)
- **Список «Выбранные файлы»** — отображает добавленные файлы
- **Кнопка «Удалить выбранный»** — удаляет выделенный файл из списка
- **Кнопка «Настройка диапазонов»** — открывает окно, где можно изменить допустимые значения Min и Max для каждого параметра (используются для раскраски и стрелок)
- **Блок «Временной интервал»** — позволяет взять данные только за нужный период:
  - Поля «Начало» и «Конец» — ввод дат вручную
  - **«Установить полный диапазон»** — программа сама найдёт самую раннюю и позднюю дату
  - **«Очистить даты»** — сбрасывает фильтр (будут взяты все данные)
- **Кнопка «Объединить файлы»** — запускает обработку

### Центральная колонка — «Выбор параметров»

Здесь ты выбираешь, **какие типы данных** нужны в отчёте (например: только «Расход», без температуры и давления).

- Список параметров берётся из файла правил, по умолчанию все выбраны
- **«Выбрать все»** / **«Снять все»** — управление всеми галочками разом

### Правая колонка — «Узлы измерения»

Здесь ты выбираешь, **с каких узлов** брать данные. Список формируется автоматически по именам добавленных файлов.

- Появляется после добавления файлов
- **«Выбрать все»** / **«Снять все»** — управление всеми галочками разом

## Пошаговая инструкция

### Шаг 1. Запуск программы

Запусти программу. Появится главное окно.

### Шаг 2. Добавление файлов

1. Нажми кнопку **«Добавить файлы»**
2. Выдели нужные файлы Excel (можно несколько сразу)
3. Нажми «Открыть» — файлы появятся в списке

> После добавления файлов программа автоматически обновит список узлов (правая колонка) и определит временной диапазон.

### Шаг 3. Выбор параметров и узлов

- В **центральной колонке** оставь галочки только на нужных параметрах
- В **правой колонке** выбери нужные узлы измерения

Если нужно всё — просто ничего не трогай, все галочки стоят по умолчанию.

### Шаг 4. Настройка времени (по желанию)

Если нужны данные только за определённый период:

1. Нажми **«Установить полный диапазон»** — программа покажет, какие даты есть
2. При необходимости скорректируй значения в полях «Начало» и «Конец»

> Формат даты: `ГГГГ-ММ-ДД ЧЧ:ММ:СС`, например `2024-01-15 08:00:00`

Если фильтр по времени не нужен — нажми «Очистить даты» или просто оставь поля пустыми.

### Шаг 5. Настройка диапазонов (по желанию)

Нажми кнопку **«Настройка диапазонов»**, чтобы проверить или изменить допустимые значения Min и Max для параметров. Эти значения влияют на цвет ячеек и стрелки в итоговом файле.

### Шаг 6. Объединение

1. Нажми кнопку **«Объединить файлы»**
2. В диалоговом окне выбери папку и введи имя для результирующего файла (например, `Отчёт_январь.xlsx`)
3. Подожди — программа обработает файлы и создаст отчёт
4. Появится сообщение «Файлы успешно объединены, создан Dashboard!»

## Что получится в итоге?

Создаётся один Excel-файл с двумя листами:

### Лист «Данные»

Полная таблица со всеми измерениями:

- **Трёхстрочный заголовок:**
  - Строка 1 — название узла (ячейки объединены для одного узла)
  - Строка 2 — допустимый диапазон: `(min ... max ед.изм.)`
  - Строка 3 — название столбца
- **Первые 3 строки и первый столбец закреплены** — они остаются видимыми при прокрутке
- **Цветовая подсветка:**
  - Белый — нет данных (значение равно 0)
  - Красный — значение **вышло** за допустимый диапазон (слишком мало или слишком много)
  - Жёлто-зелёный градиент — значение **в норме** (чем ближе к максимуму, тем зеленее)
- **Столбцы-индикаторы ⚠** рядом с каждым параметром:
  - `↓` — значение меньше нормы
  - `↑` — значение больше нормы
  - Пусто — значение в норме или данных нет

### Лист «Dashboard»

Сводная таблица по дням — сразу видно общую картину:

| Узел | Допустимый диапазон | Выход за диапазон | 01.01.2024 | 02.01.2024 | ... |
|------|---------------------|-------------------|------------|------------|-----|
| | Итого: | 5 сут.; 12 ч. | 450.32 | 423.10 | |
| Узел 1 | (10 ... 100 тыс.м3/ч) | 2 сут.; 5 ч. | 125.44 | 8.21 | |

**Цвета ячеек на Dashboard:**

| Цвет | Значение |
|------|---------|
| Зелёный | Данные в норме |
| Красный | Есть нарушения допустимого диапазона |
| Жёлтый | Расход слишком мал (≤ 12 тыс. м³ в сутки) |
| Серый | Нет данных (все значения = 0) |

В каждой ячейке дашборда отображается суммарный расход за день, а поверх него нарисован мини-график (спарклайн) с трендом изменений в течение суток.

## Частые вопросы

**Нажимаю «Объединить», но появляется ошибка или ничего не происходит**

Проверь:
1. Добавлен ли хотя бы один файл?
2. Выбран ли хотя бы один параметр и один узел?
3. Не открыт ли итоговый файл в Excel? Закрой все окна Excel и повтори попытку.
4. Есть ли в выбранных файлах данные за указанный временной интервал?

**В итоговом файле нет нужных параметров**

В центральной колонке посмотри — возможно, галочка напротив нужного параметра снята. Или этих данных нет ни в одном из выбранных файлов.

**Правая колонка с узлами пустая**

Программа определяет узлы по именам файлов, сверяясь с файлом правил. Убедись, что имена файлов соответствуют правилам. Если нет — обратись к администратору или автору программы.

**Что такое файл «Правила названия столбцов.xlsx»?**

Это «мозг» программы. В нём записано, как переименовывать столбцы из каждого файла в единые стандартные названия, а также допустимые диапазоны (Min/Max) для каждого параметра.

> **Не удаляй и не перемещай этот файл** — без него программа не сможет работать.

**Как изменить допустимые значения (Min и Max)?**

Нажми кнопку «Настройка диапазонов» в левой колонке. Измени нужные значения и нажми «Сохранить». После сохранения изменения вступят в силу при следующем объединении.
